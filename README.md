#### Relatórios Visuais (Dashboards) - Power BI
Relatórios visuais feitos no Power BI para apresentações Institucionais. Foram utilizados dados de demanda e venda de animais tanto para a UFMG quanto para instituições externas. <br><br>
**Observação:** Estão mostrados apenas os dashboards e a apresentação autorizada pela Coordenação do Biotério Central. Na apresentação, também é possível visualizar um dashboard antigo de dados financeiros. Por fim, no final da apresentação há um relatório visual cruzando dados de demanda e produção e que foram cruciais para ajustar a produção e, destarte, minimizar custos.
